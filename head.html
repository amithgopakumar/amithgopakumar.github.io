Head Controlled Mouse

People with severe motor disabilities require a variety of options to access computers and alternative communication technologies. Tracking the head movement of the user to control the mouse cursor can be done in several ways - using a webcam or using sensors. Body-worn devices with gyroscope sensor that can track the head movement are available commercially as Assistive Technology (AT). Although these devices give greater independence to the users, they are very expensive and not affordable in a developing country like India.

Searching on the internet, how to make the head controlled device to operate the computer, some DIY pages gives an idea that, every one can simply make a computer mouse using an Arduino development board. Referring to the Fablab documentation of Akila Surendran ( Senior Engineer in CATI ) give some more idea about how to consider the user needs also.

Things used in this Project

Hardware Components

Atmega32U4 x 1:Atmega32U4 microchip based board is used for developing this project, Atmega32U4 is a low-power Microchip 8-bit AVR RISC-based microcontroller featuring 32KB self-programming flash program memory, 2.5KB SRAM, 1KB EEPROM, USB 2.0 full-speed/low-speed device, 12-channel 10-bit A/D-converter, and JTAG interface for on-chip-debug. Compare to other development boards like Arduino Uno Atmega32u4 have the ability to be an HID controller at the same time as being able to load the firmware via USB.

MPU6050 x 1:MPU6050 is a Micro-Electro-Mechanical Systems which consists of a 3-axis Accelerometer and 3-axis Gyroscope inside it. This helps us to measure acceleration, velocity, orientation, displacement and many other motion-related parameters of a system or object.

Software Components

Arduino IDE : For programming the microcontroller board

Fusion 360 : For Design the case for holding the sensor and microcontroller

Story

I along with Akila from NISH and Varun intern from RET started the intial prototyping work of Head mouse.Head mouse with MPU6050 and Arduino Micro’ is the Instructables page that we have referred to.Further refinements to the available code had to be done to decrease the sensitivity of the gyroscope module and to increase the sensitivity of the pointer so that noise from the gyroscope is neglected to avoid the ‘ghost movement’ of the pointer. The casing for the head mouse was designed using Fusion 360 and 3D-printed in Ultimaker 2. To mount the head mouse over the head and hold it intact, we used the components from a headphone. More details can be found in the Github page in developing this head-controlled mouse

Once initial Prototype is completed , we tested the head-controlled mouse with around 10 able-bodied individuals who tried both the free camera mouse software and our DIY head-controlled mouse..Everyone preferred the DIY head-controlled mouse over the camera mouse software for user-friendliness and precision in tracking and gives feedback and suggestion to head mouse. Considering the Feedback and Sujjestion from them, we designed a small and compactable modelwith new firmware.

Final Prototype

Arduino micro is a large board and if we are using headmouse size become quite large, so my first task is to make it smaller. I go through different electronic shops various websites and finally I found a small development board with normal USB Type A, it’s compactable for our final prototype.MPU6050 sensor board and Development board has almost the same size It gives a big relief to me, our product size is going to be smaller than the first prototype.

After the purchase of the components, next design the outercase for HeadMouse, First I draw sample sketch in paper and then take measurements using a digital vernier calliper. Using the measurements , designed the case for holding the Board and sensor for the head mouse. Fusion 360 cad software is used for design the case.After design , files are exported as STL and get it printed from the 3D printer
